## 例题精讲

### 【题型1】统计量与枢轴量的判断

**判断标准**：
- **统计量**：样本的函数，**不含未知参数**
- **枢轴量**：含样本和未知参数，但其**分布不依赖于未知参数**

---

**例1（作业3 Problem 1）**：$X_1,\ldots,X_n \sim U[0,\theta]$，$G = \bar{X}/\theta$。判断是统计量还是枢轴量。

**解题思路**：
1. $G$ 含未知参数 $\theta$，不是统计量
2. 检查分布：$X_i/\theta \sim U[0,1]$，故 $G = \dfrac{1}{n}\sum \dfrac{X_i}{\theta}$ 服从某个与 $\theta$ 无关的分布
3. → **枢轴量** ✓

**知识点**：枢轴量的关键是分布与参数无关，不是表达式与参数无关。

---

**例2**：$X_1,\ldots,X_n \sim N(\mu,\sigma^2)$，$\mu,\sigma$ 均未知，$G = \dfrac{(n-1)S^2}{\sigma^2}$。判断类型。

**解题思路**：
1. 含未知参数 $\sigma^2$，不是统计量
2. 由学生定理：$G \sim \chi^2(n-1)$，分布与 $\mu,\sigma^2$ 均无关
3. → **枢轴量** ✓

---

### 【题型2】MLE求解

**通用步骤**：写 $l(\theta)$ → 求导令为0 → 注意顺序统计量特殊情况（支撑含参数时）

---

**例3（作业2 Problem 9）**：$f(x;\lambda) = \dfrac{4x^3}{\lambda^4}$（$0 < x \le \lambda$），求 $\lambda$ 的MLE。

**解题思路**：
1. 似然函数 $L(\lambda) = \dfrac{\prod 4x_i^3}{\lambda^{4n}}$，当 $\lambda \ge X_{(n)}$ 时有效
2. $L(\lambda)$ 关于 $\lambda$ **单调递减**（分母 $\lambda^{4n}$ 越小越好）
3. → $\hat{\lambda} = X_{(n)} = \max X_i$

**无偏纠正**：计算 $E(X_{(n)})$，积分求出偏差，乘以修正系数使之无偏。

---

**例4（作业5 Problem 1）**：瑞利分布 $f(x;\sigma) = \dfrac{x}{\sigma^2} e^{-x^2/(2\sigma^2)}$，$x>0$，求 $\hat{\sigma}$ 的MLE。

**解题思路**：

$$l(\sigma) = \sum\ln x_i - 2n\ln\sigma - \frac{1}{2\sigma^2}\sum x_i^2$$

$$\frac{dl}{d\sigma} = -\frac{2n}{\sigma} + \frac{1}{\sigma^3}\sum x_i^2 = 0 \quad\Rightarrow\quad \hat{\sigma} = \sqrt{\frac{\sum x_i^2}{2n}}$$

代入数据 $\sum x_i^2 = 47.82$，$n=10$：$\hat{\sigma} = \sqrt{47.82/20} \approx 1.546$

### 【题型3】置信区间构造

**模板**（非对称参数如 $\sigma^2$ 需注意两个分位数方向）：

---

**例5（作业3 Problem 3）**：$X \sim N(\mu,\sigma^2)$，$n=100$，$\bar{x}=15.5$，$s=3.6$，求：
(1) $\mu$ 的95%双侧置信区间；(2) $\sigma^2$ 的99%单侧置信上界。

**解题思路**：

(1) $\sigma^2$ 未知，用 $t$ 检验，枢轴量 $G = \dfrac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(99)$

$$CI_\mu = \left(15.5 - t_{0.025}(99)\cdot\frac{3.6}{\sqrt{100}},\ \ 15.5 + t_{0.025}(99)\cdot\frac{3.6}{\sqrt{100}}\right) = \left(15.5 \pm 0.36\cdot t_{0.025}(99)\right)$$

(2) 单侧置信**上界**，枢轴量 $G = \dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(99)$

$$P\!\left(\frac{99S^2}{\sigma^2} > G_0\right) = 0.99 \quad\Rightarrow\quad G_0 = \chi^2_{0.99}(99)$$

$$\sigma^2 < \frac{99S^2}{\chi^2_{0.99}(99)} = \frac{99 \times 12.96}{\chi^2_{0.99}(99)} = \frac{1283.04}{\chi^2_{0.99}(99)}$$

### 【题型4】假设检验完整流程（6步法）

**标准流程**：① 建立假设 → ② 选统计量 → ③ 写拒绝域形式 → ④ 计算临界值 → ⑤ 代入数据 → ⑥ 作出结论

---

**例6（作业4 Problem 3）**：某校学生AI使用时长 $\mu_0 = 15$ h，$\sigma=4$，$n=64$，$\bar{x}=16.2$，$\alpha=0.05$，检验是否高于全国水平。

**解**：
1. $H_0: \mu=15$ vs $H_1: \mu>15$（右侧检验）
2. $\sigma$ 已知，用 $z$ 检验：$Z = \dfrac{\bar{X}-\mu_0}{\sigma/\sqrt{n}} \sim N(0,1)$
3. 拒绝域：$\{Z \ge z_{0.05} = 1.645\}$
4. $z = \dfrac{16.2-15}{4/\sqrt{64}} = \dfrac{1.2}{0.5} = 2.4$
5. $2.4 > 1.645$，拒绝 $H_0$
6. 结论：在5%水平下，该校学生AI使用时长显著高于全国平均水平。

**p值**：$p = P(Z \ge 2.4) = 1 - \Phi(2.4) \approx 0.0082 < 0.05$，仍拒绝 $H_0$

---

**例7（作业4 Problem 5，成对数据）**：10名同学专注模式前后焦虑得分差 $D_i = X_i - Y_i$，$\bar{d}=1.5$，$s_D \approx 4.403$，$\alpha=0.05$，检验是否有效。

**解**：
1. $H_0: \mu_D=0$ vs $H_1: \mu_D>0$
2. 成对 $t$ 检验：$T = \dfrac{\bar{D}}{S_D/\sqrt{n}} \sim t(9)$
3. $t = \dfrac{1.5}{4.403/\sqrt{10}} = \dfrac{1.5}{1.392} \approx 1.077$
4. $t_{0.05}(9) = 1.833$，$1.077 < 1.833$，**不拒绝** $H_0$
5. 结论：没有足够证据表明专注模式能显著降低焦虑。

**注意**：成对检验消除了个体间差异，比独立样本检验更精确。

### 【题型5】ANOVA方差分析表

**口诀**：$S_T = S_A + S_E$；$F = MS_A / MS_E$；自由度：组间 $r-1$，组内 $n-r$

---

**例8（作业4 Problem 8）**：3种烘焙程度，$n_1=21, n_2=31, n_3=26$，$S_1^2=0.45, S_2^2=0.38, S_3^2=0.52$，$S_A=5.40$，$\alpha=0.05$。

| 来源 | SS | df | MS | F | p |
|------|----|----|-----|---|---|
| 组间 | 5.40 | 2 | 2.70 | 6.063 | 0.0036 |
| 组内 | $S_E = \sum(n_i-1)S_i^2 = 33.4$ | 75 | 0.4453 | | |
| 总计 | 38.80 | 77 | | | |

$F = 2.70/0.4453 \approx 6.063 > F_{0.05}(2,75) \approx 3.12$，**拒绝** $H_0$，不同烘焙程度对评分有显著影响。

### 【题型6】Fisher信息量计算（三种方法）

**最推荐方法三（负二阶导）**：对 $\log f$ 求二阶导，取负期望。

---

**例9**：$X \sim \text{Bernoulli}(\theta)$，求 $I(\theta)$。

**方法三**：

$$\log f = x\log\theta + (1-x)\log(1-\theta)$$

$$\frac{\partial^2\log f}{\partial\theta^2} = -\frac{x}{\theta^2} - \frac{1-x}{(1-\theta)^2}$$

$$I(\theta) = -E\!\left[-\frac{X}{\theta^2} - \frac{1-X}{(1-\theta)^2}\right] = \frac{\theta}{\theta^2} + \frac{1-\theta}{(1-\theta)^2} = \frac{1}{\theta} + \frac{1}{1-\theta} = \frac{1}{\theta(1-\theta)}$$

---

**例10**：$X \sim \text{Poisson}(\theta)$，写出三大检验统计量（$H_0: \theta = \theta_0$）。

| 检验 | 统计量 | 注意点 |
|------|--------|--------|
| LRT | $\chi^2_L = 2n\left[\bar{X}\log(\bar{X}/\theta_0) - (\bar{X}-\theta_0)\right]$ | 含 $\log$ |
| Wald | $\chi^2_W = n(\bar{X}-\theta_0)^2 / \bar{X}$ | 分母用 $\hat{\theta}=\bar{X}$ |
| Score | $\chi^2_R = n(\bar{X}-\theta_0)^2 / \theta_0$ | 分母用 $\theta_0$ |

### 【题型7】充分统计量（因子分解定理）

**步骤**：写出联合密度 → 分离含 $\theta$ 的部分 → 含 $\theta$ 的函数的参数即CSS

---

**例11（作业6 Problem 4）**：各分布的充分统计量汇总（精选）

**(1)** $P(X=k) = 1/\theta$，$k=1,\ldots,\theta$：

$$\text{联合分布} \propto (1/\theta)^n \cdot \mathbf{1}\{X_{(n)} \le \theta\} \quad\Rightarrow\quad \text{CSS} = X_{(n)}$$

**(2)** $f(x;\theta) = \dfrac{\theta}{x^{\theta+1}}$，$x>1$：

$$\log f = \log\theta - (\theta+1)\log x \quad\Rightarrow\quad \text{CSS} = \sum\log X_i$$

**(3)** Laplace分布 $f(x;\theta) = \dfrac{1}{2\theta}e^{-|x|/\theta}$：

$$\text{联合密度} \propto \theta^{-n}\exp\!\left\{-\frac{1}{\theta}\sum|x_i|\right\} \quad\Rightarrow\quad \text{CSS} = \sum|X_i|$$

### 【题型8】MVUE：Rao-Blackwell条件期望法

---

**例12（作业6 Problem 8，Bernoulli）**：$X_i \sim \text{Bernoulli}(\theta)$，求 $g(\theta) = (1-\theta)^2$ 的MVUE。

**步骤**：
1. CSS：$Y_1 = \sum X_i \sim \text{Bin}(n,\theta)$（正则指数族）
2. 无偏估计：$Y_2 = \mathbf{1}\{X_1+X_2=0\}$，$E(Y_2) = (1-\theta)^2$ ✓
3. 计算 $E(Y_2 \mid Y_1=t) = P(X_1=0, X_2=0 \mid Y_1=t)$
4. 超几何分布：

$$= \frac{\dbinom{n-t}{2}}{\dbinom{n}{2}} = \frac{(n-t)(n-t-1)}{n(n-1)}$$

5. → MVUE $= \dfrac{(n-\sum X_i)(n-\sum X_i-1)}{n(n-1)}$

---

**例13（作业6 Problem 7，几何分布）**：$X_i \sim \text{Geometric}(\theta)$，求 $\theta$ 的MVUE。

1. CSS：$T = \sum X_i \sim \text{Pascal}(n,\theta)$
2. 无偏估计：$\psi(X_1) = \mathbf{1}\{X_1=1\}$，$E(\psi) = \theta$ ✓
3. 条件期望：

$$E(\psi \mid T=t) = P(X_1=1 \mid T=t) = \frac{\dbinom{t-2}{n-2}}{\dbinom{t-1}{n-1}} = \frac{n-1}{t-1}$$

4. → MVUE $= \dfrac{n-1}{\sum X_i - 1}$

## 综合题提示

**遇到综合题的解题顺序**：
1. 看分布 → 判断是否属于指数族（支撑是否含参数）
2. 找CSS（指数族用 $\sum K(X_i)$，均匀分布用顺序统计量）
3. MLE → 检查无偏性 → 与RCB比较（有效性）
4. 若要MVUE → L-S直接法（猜测MLE然后纠偏）或R-B条件期望法
